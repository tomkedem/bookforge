# Examples of ML Features (or ML Characteristics)


| Feature | Technical/Business Explanation |
| --- | --- |
| Speech to text | Converting sound waves to written text in real-time or retroactively. |
| Semantic Categories | Classifying sentences by meaning (topics) and not just by keywords (for example: recognizing that "I want to leave", "I want to end the contract" belongs to the "technical support" category). |
| Sentiment Analysis | Identifying the customer's emotional state (positive, negative, neutral) based on tone of speech and word choice. |
| Call Summaries | Using GenAI to create an automatic summary of the call, saving the agent the need to type a manual summary. |
| Prompt Customization | The ability to define custom instructions for language models (LLMs) to get more accurate results for the organization. |
| Automatic Quality Management | Automatic control over all calls (instead of sample listening) to ensure compliance with procedures and service quality. |
| AI Insights - Automatic Call Analysis | Extracting deep conclusions from all interactions without human touch. |
| Discovery - Intent / Resolution / Clustering | Identifying what the user wanted (Intent), whether the problem was solved (Resolution), grouping similar calls into groups (Clustering) to identify new trends, understanding what's in the calls. |



**Points to Consider:**

• **Discovery (Clustering):**

A particularly powerful tool. Instead of telling the system what to look for, the system uses Unsupervised Learning to tell you: "Pay attention, you suddenly have 200 calls about a problem you didn't know about before."

• **Prompt Customization:**

Today, a significant part of the Logic of these products moves from hard-coded to managing complex prompts against models like GPT or Gemini.

• **Intent vs. Resolution:**

The ability to distinguish between what the customer wanted and whether they received a response is the main measure for ROI of such systems.



In the world of Machine Learning, the word **Features** carries a double meaning, and both interpretations are relevant to the list we presented in the table:

1. **Features (in the product sense - Product Features):**

Functional capabilities that the system offers to the end user thanks to artificial intelligence (such as: transcription or call summaries). This is the most likely meaning of the list title.

2. **Features (in the technical sense - Model Features):**

The variables or data used as input to the mathematical model (such as: call length, specific words spoken, or tone of speech) to produce the prediction.
