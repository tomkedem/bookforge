# Casos de Uso Comunes


| Escenario | Significado Empresarial/Técnico |
| --- | --- |
| Llamadas Repetidas | Identificar clientes que llaman una y otra vez sobre el mismo tema. El cliente probablemente no está satisfecho y por eso llama repetidamente. Si el 20% de las llamadas son repetidas, pueden ahorrar mucho dinero. Objetivo: mejorar el FCR (Resolución en Primera Llamada) y así ahorrar dinero. |
| Llamadas Largas | Análisis de llamadas inusualmente largas para entender dónde se retrasó el agente o dónde el proceso es demasiado engorroso. |
| Autoservicio | Identificar temas que pueden transferirse a bots o al área personal en el sitio web para reducir la carga en el centro de llamadas humano y ahorrar costos. |
| Abandono de Clientes | Analizar señales de advertencia temprana en la conversación (como amenazas de irse o insatisfacción extrema) para evitar que el cliente se vaya. Es como un globo - por un lado aumento el aire, hay clientes que se unen, y por el otro lado disminuyo el aire, hay clientes que se van, y quiero aumentar el aire en el globo. |
| Sentimiento (o Estado de Ánimo) | Medir el nivel de felicidad o enojo del cliente durante la llamada. |
| Ventas | Identificar oportunidades de ventas adicionales (Upsell) o analizar técnicas de cierre de agentes destacados. |
| Mejora (Rendimiento/Procesos) | Término general para optimización del sistema, capacitación de agentes y eficiencia del centro de llamadas. |



**Análisis Profesional:**

1. **Llamadas Repetidas y Llamadas Largas:** Muy fácil crear dashboards sobre ellas que muestren valor inmediato (ROI) a la gerencia.

2. **Optimización del Autoservicio:** Aquí es donde entran el NLP y el clustering de datos. El sistema identifica qué preguntas se repiten lo suficiente como para que valga la pena desarrollar automatización para ellas.

3. **Predicción de Abandono:** Este es uno de los escenarios de ML más complejos. Requiere Enriquecimiento de Datos entre el texto de la llamada y los datos del CRM del cliente (cuánto tiempo lleva en la empresa, cuánto paga, etc.).
