# מבוא

השיעור מציג את היסודות של Machine Learning בתוך ההקשר הרחב של Artificial Intelligence, ומטרתו לבנות הבנה הדרגתית של התחום.

תפקידינו כ-AI Engineers וכדימוי, הוא לקחת את המנוע ולבנות ממנו מכונית. וכדי לעשות זאת, עלינו להבין את המעטפת.

**נלמד:**

- מה זה בכלל AI, איפה Machine Learning משתלב בתוכו, ולמה משתמשים בו

- איך נראית התמונה הגדולה עוד לפני שנכנסים לאלגוריתמים עצמם

- איך להשתמש במודלים קיימים ולהטמיע אותם במוצרים שיעבדו בפרודקשיין

המטרה כאן אינה ללמוד נוסחאות או מימושים, אלא להבין איך לחשוב נכון על Machine Learning.

התחום נראה מורכב במבט ראשון, אבל ברגע שמבינים את התמונה הגדולה, הכול מתחיל להתחבר.
