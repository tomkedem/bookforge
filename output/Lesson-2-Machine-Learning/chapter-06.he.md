# איך עובד תהליך Machine Learning בפועל

Machine Learning הוא תהליך שבו המערכת לומדת מתוך נתונים,

כדי לבצע תחזיות על נתונים חדשים.

<img src="/Lesson-2-Machine-Learning/assets/image-06.png" alt="image-06.png" width="710" height="293" />


**התהליך מורכב מארבעה שלבים מרכזיים:**

- מתחילים מאיסוף נתונים

- מחלקים את הנתונים לאימון ולבדיקה

- מאמנים מודל לזהות דפוסים בנתונים

- משתמשים במודל כדי לחזות תוצאות עבור נתונים חדשים

**המפתח להבנת כל התהליך הוא זה:** 
המודל לא מקבל חוקים מוכנים מראש, אלא לומד אותם מתוך הנתונים.
