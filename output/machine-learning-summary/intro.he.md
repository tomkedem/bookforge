# פתיחה

השיעור מציג את היסודות של Machine Learning בתוך ההקשר הרחב של Artificial Intelligence, ומטרתו לבנות הבנה הדרגתית של התחום.

נבין מה זה בכלל AI, איפה Machine Learning משתלב בתוכו, למה משתמשים בו, ואיך נראית התמונה הגדולה עוד לפני שנכנסים לאלגוריתמים עצמם.

המטרה כאן אינה ללמוד נוסחאות או מימושים, אלא להבין איך לחשוב נכון על Machine Learning.

התחום של Machine Learning נראה מורכב במבט ראשון, אבל ברגע שמבינים את התמונה הגדולה, הכול מתחיל להתחבר.
