# מה זה Machine Learning

Machine Learning הוא תחום בתוך AI שמאפשר למערכות ללמוד ולהשתפר מתוך ניסיון, בלי להיות מתוכנתות במפורש.

המשמעות היא שהמערכת לא עובדת לפי חוקים קבועים מראש, אלא לומדת דפוסים מתוך נתונים ומשתפרת בהתאם.

<img src="../assets/image-02.png" alt="image-02.png" width="385" height="577" />

בנוסף, התחום מתמקד בפיתוח תוכנות שיודעות לגשת לנתונים ולהשתמש בהם כדי ללמוד בעצמן.

להלן מוצגת היררכיה של התחום:

AI

→ Machine Learning

→  Neural Networks

→  Deep Learning

→  Generative AI / NLP / Large Language Models

היררכיה זו מראה שמדובר ברמות שונות של התמחות, כאשר כל שכבה היא תת תחום של השכבה שמעליה.
