# Supervised Learning

<img src="../assets/image-09.png" alt="image-09.png" width="455" height="240" />

In supervised learning, the model learns from data that has known answers in advance.

The data comes with labels
That is, each example has a correct answer

The algorithm learns to identify the relationship between the data and the outcome
And later uses what it learned to predict results for new data

The workflow:

Starting from labeled data
The model trains on them and learns the patterns

Then new unlabeled data is received
And the model provides a prediction or classification

The output is the result that the model estimates
For example, identifying a type of fruit from an image
