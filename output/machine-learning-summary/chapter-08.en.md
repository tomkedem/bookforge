Machine Learning is divided into two main types of learning:

Supervised Learning

There is data with known answers in advance, meaning we know what the correct result is that the model should learn

In this situation, there are two types of problems:

- When the output is a number, we perform prediction of a continuous value

- When the output is a category, we perform classification into groups

Unsupervised Learning

There are no known answers in advance. The model receives only data and tries to find structure in it

In this situation, there are two main types:

- Clustering: dividing the data into similar groups

- Associations: identifying relationships between items within the data

Supervised Learning

In supervised learning, the model learns from data that has known answers in advance.

The data comes with labels
That is, each example has a correct answer

The algorithm learns to identify the relationship between the data and the outcome
And later uses what it learned to predict results for new data

The workflow:

Starting from labeled data
The model trains on them and learns the patterns

Then new unlabeled data is received
And the model provides a prediction or classification

The output is the result that the model estimates
For example, identifying a type of fruit from an image
