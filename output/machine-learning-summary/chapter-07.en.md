# General Machine Learning Process

Machine Learning works as an organized process that repeats itself.

Starting from data

There is data that is divided into training and testing

The data goes to algorithms

The algorithms search for patterns within the data

From this learning, a model is created

The model is a representation of what was learned from the data

Then the model is used on new data

![image-14.png](../assets/image-14.png)

To perform prediction, classification, or clustering

The process doesn't end here

There is repetition of the training and testing phase to improve the model

Important point

The model doesn't store the data itself

But rather the patterns learned from it
