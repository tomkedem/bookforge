# What is Machine Learning

Machine Learning is a field within AI that enables systems to learn and improve from experience, without being explicitly programmed.

This means the system doesn't work according to fixed, predefined rules, but learns patterns from data and improves accordingly.

Additionally, the field focuses on developing software that knows how to access data and use it to learn on its own.

The following hierarchy of the field is presented:

AI

→ Machine Learning

→  Neural Networks

→  Deep Learning

→  Generative AI / NLP / Large Language Models

This hierarchy shows different levels of specialization, where each layer is a sub-field of the layer above it.
