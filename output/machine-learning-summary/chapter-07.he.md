# תהליך כללי של Machine Learning

Machine Learning עובד כתהליך מסודר שחוזר על עצמו.

מתחילים מנתונים

יש נתונים שמחולקים לאימון ולבדיקה

הנתונים עוברים לאלגוריתמים

האלגוריתמים מחפשים דפוסים בתוך הנתונים

מהלמידה הזו נוצר מודל

המודל הוא ייצוג של מה שנלמד מהנתונים

לאחר מכן משתמשים במודל על נתונים חדשים
