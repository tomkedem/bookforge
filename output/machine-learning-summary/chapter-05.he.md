# שימושים קלאסיים של Machine Learning

<img src="../assets/image-04.png" alt="image-04.png" width="427" height="392" />

Machine Learning נכנס להרבה מאוד מערכות בעולם האמיתי.

יש מערכות שמטרתן לזהות מצבים חריגים

כגון: ניסיונות הונאה או פריצה לרשת. המערכת לומדת מה נחשב התנהגות רגילה, וכאשר משהו נראה שונה, היא מזהה זאת כחשוד.

יש מערכות שמטרתן להציג למשתמש את התוכן הכי מתאים לו

כגון: תוצאות חיפוש, פרסומות או המלצות על מוצרים. כאן המערכת לומדת מהעדפות המשתמשים ומשפרת את ההתאמה.

יש שימושים בעולם העסקי

כגון:הערכת סיכון של לקוח או קביעת מחיר. המערכת מנתחת נתונים כדי לעזור בקבלת החלטות.

יש גם מערכות שמטרתן לחזות תקלות מראש

למשל במערכות מחשוב, כדי לטפל בבעיה לפני שהיא מתרחשת.

בנוסף, יש שימושים שבהם המערכת מנתחת מידע מורכב כמו טקסטים ותמונות, ומנסה להבין את המשמעות שלהם.

בסופו של דבר, כל הדוגמאות האלו מתבססות על אותם רעיונות מרכזיים: לזהות, לחזות, לדרג ולהתאים.
